# Quantium Virtual Internship — Retail Strategy & Analytics: Task 1

## Overview
This project completes Task 1 of the Quantium Data Analytics Virtual 
Internship (Forage). It covers full data preparation and customer 
analytics on retail chip transaction data across 264,836 records.

## What Was Done
- Cleaned and validated two large-scale QVI datasets (transactions + customer data)
- Resolved data quality issues including outliers, missing dates, and brand inconsistencies
- Engineered new features: pack size, brand name, price per unit
- Performed customer segment analysis by lifestage and premium tier
- Conducted brand and pack size affinity analysis for Mainstream Young Singles/Couples
- Applied Welch t-test to confirm statistically significant price differences (p < 2.2e-16)

## Key Findings
- Budget Older Families, Mainstream Young Singles/Couples, and Mainstream 
  Retirees are the top sales-driving segments
- Mainstream Young Singles/Couples are 23% more likely to buy Tyrrells chips
- Mainstream Young Singles/Couples are 27% more likely to buy 270g packs
- Mainstream young/mid-age singles and couples pay significantly more per unit 
  than Budget and Premium counterparts
